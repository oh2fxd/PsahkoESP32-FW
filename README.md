### How to Flash
1. Download the firmware file
2. Use [esptool.py](https://github.com/espressif/esptool) or ESP32 Flash Download Tool
3. Set flash mode: QIO, Flash size: 4MB, Flash freq: 80MHz
4. Upload at address 0x10000
